Инструкция:

Редактируем config.py

Добавляем API ID и API HASH (где и как регистрироваться знают все):

nano config.py

api_id = ваш api_id
api_hash 'ваш api_hash'


Как добавить текст:

text = ['ваш текст', 'ваш текст'] и т.д


Добавляем аккаунт:

python3 sessions.py

Вводим номер телефона
Вводим код

Запускаем:

python3 bot.py
